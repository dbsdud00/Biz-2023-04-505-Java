- 내가 요청한 데이터는 테이블 형태로 보여줌

### 1) 요청의 종류(DB 4대 업무 crud)
- 데이터 추가 : create

- 데이터 읽기 : read
  - 명령어 : select
- 데이터 변경(수정) : update
  - 명령어 : update
- 데이터 삭제 : delete
