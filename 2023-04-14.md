### 컴파일 (Compile)
- Text type 으로 작성된 소스코드를 컴퓨터에 주입하여 실행하기 위해 BIN(이진수) 코드로 한꺼번에 변환하는 것
### 컴파일러 (Compiler)
- 소스코드를 컴파일 하여 BIN 코드를 만들어 내는 소프트웨어 개발 도구
### 인터프리터 (Interpreter)
- Text Type 으로 작성된 소스코드를 한줄씩 실행
