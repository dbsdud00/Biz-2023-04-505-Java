## vi
- Editor : 파일을 생성하고 수정하고 (내용)삭제할 수 있는 도구
- text Editor : Editor중 텍스트를 사용하는 Editor - vi
- 키보드만 가지고 데이터 입력
- 입력상태와 명령상태의 두가지 모드
- 시작되면 명령 모드로 시작됨
- 명령 a : 입력상태로 변경
- ESC : 명령상태로 이동
- 명령 ':w' : 저장
